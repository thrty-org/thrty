## A taste

```typescript
import type { Handler } from 'aws-lambda';
import { compose, typesOf } from '@thrty/core';
import { inject } from '@thrty/inject';
import { errorHandler } from '@thrty/error-handler';
import { validate } from '@thrty/validator';
import { z } from 'zod';

const Notification = z.object({
  id: z.string(),
  amount: z.number(),
  recipient: z.string().email(),
});

export const handler = compose(
  typesOf<Handler<unknown>>(),
  inject({ notifications: notificationsFactory }),
  errorHandler(),
  validate(Notification),
)(async (event, context) => {
  await context.deps.notifications.send(event);
});
```

Every property the actual handler reaches for is contributed by a middleware higher up the chain — `context.deps.notifications` from `inject`, the typed `event` from `validate`. The handler's signature is inferred from the whole chain, not annotated by hand.
